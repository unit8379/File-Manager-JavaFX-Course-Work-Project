# File-Manager-JavaFX-Course-Work-Project

Кроссплатформенный файловый менеджер, который представляет собой оконное приложение.
В качестве дополнительного функционала может порождать дочерний процесс и общаться с ним через IPC с применением Memory-Mapped Files.

Программа написана с применением Java 14.0.1 и JavaFX 14.0.1, а также использована библиотека jProcesses для распарсинга информации о процессах из консоли (для доп. задания курсовой работы).
